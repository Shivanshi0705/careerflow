# CareerFlow

CareerFlow is a modern job application and networking tracker designed to help users stay organized throughout the recruiting process. It allows users to manage applications, monitor pipeline stages, track recruiter and networking contacts, and view key dashboard insights in one place.

## Overview

The goal of CareerFlow is to simplify internship and job searching by giving users a clean, centralized system to:

- track submitted applications
- manage recruiting stages like Applied, OA, Interview, Final Interview, Offer, and Rejected
- store networking contacts and follow-up details
- monitor recent activity and quick performance metrics
- keep role-specific notes, resume versions, and cover letter versions together

This project was built with a strong focus on usability, organization, and modern UI design.

## Features

### Application Tracking
- Add and manage internship/job applications
- Track company, role, location, job link, date applied, and current status
- Update statuses directly from the applications page
- Sort applications by latest applied
- View pipeline progress across multiple stages

### Dashboard Analytics
- Total applications count
- Applied, interview, offer, and rejected metrics
- Interview rate and offer rate
- Recent activity across tracked applications
- Application status breakdown
- Next actions panel for important follow-ups

### Contact Management
- Add and manage recruiter, mentor, and referral contacts
- Store company, role, email, LinkedIn, and notes
- Track last contact date
- Highlight newest contacts first
- Show recent contacts on the dashboard

### Documents and Notes
- Save resume version and cover letter version used for each application
- Upload resume and cover letter PDFs
- Add custom notes for each application

### Authentication
- Demo login flow for testing the app
- Local storage-based user session handling

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Lucide React
- LocalStorage for mock persistence

## UI / Design Goals

CareerFlow was designed with a modern dark theme and a dashboard-first experience. The interface focuses on:

- clean spacing and card-based layout
- strong visual hierarchy
- simple navigation
- clear status visibility
- a polished recruiting-focused workflow
